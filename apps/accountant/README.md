# Receipt24 Accountant Portal

Secure portal for accountants and accounting firm managers.

**Status:** Scaffold only (Phase 1–2). Portal UI begins in Phase 9 (auth screens may appear earlier).

## Planned capabilities

- Registration + professional verification upload
- Client invitations and access scopes
- Client workspace, classification, notes, document requests
- CSV / Excel / PDF / ZIP exports

## Access rule

Only clients with explicit consumer approval are visible.
