# Receipt24 Consumer App

Flutter mobile-first consumer application.

**Status:** Scaffold only (Phase 1–2). UI begins in Phase 3.

## Planned capabilities

- Auth + onboarding
- Receipt capture (camera, image, PDF, email import, manual)
- Wallet, insights, warranties, returns
- Accountant sharing
- Subscriptions and privacy settings

## Suggested stack

- Flutter
- `supabase_flutter`
- Shared constants from `packages/shared` (mirrored Dart types in later phase)

## Do not implement

- Merchant registration, login, dashboards, QR issuing, or refund processing
