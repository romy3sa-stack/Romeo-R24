# Consumer App area

Screens for the Consumer App (Phase 3 onboarding/auth, Phase 4 dashboard,
Phase 5 capture, Phase 6 expense management, Phase 7 insights, Phase 8
warranties/returns) live here. Intentionally empty in Phase 1/2 — see the
root instruction: "Do not build dashboards yet."

Routing target: `AppArea.consumer` (`lib/core/routing/app_area.dart`), reached
by users with `UserRole.consumer`.
